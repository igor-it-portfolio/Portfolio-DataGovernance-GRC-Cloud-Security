# 🛠️ Technical Foundations & Automation Controls

**PT-BR:**
Este repositório reúne projetos técnicos focados em **Fundamentos de Infraestrutura, Automação com Scripts, Controles de Segurança e Rotinas Operacionais** que sustentam ambientes de dados e BI.

Aqui apresento projetos práticos utilizando **Linux, Bash Scripting, Hardening de Sistemas, Automação de Backups, Monitoramento de Integridade e Registros de Auditoria**. Essa base técnica garante a resiliência, a segurança e a auditabilidade das plataformas onde as bases de dados e rotinas analíticas são armazenadas e processadas.

**EN:**
This repository features technical projects focused on **Infrastructure Foundations, Scripting Automation, Security Controls, and Operational Routines** that support modern data and analytics environments.

Here I demonstrate practical hands-on projects using **Linux, Bash Scripting, System Hardening, Backup Automation, Integrity Monitoring, and Audit Logging**. This technical foundation ensures the resilience, security, and auditability of the underlying systems where enterprise data is stored and processed.

## 🛠️ Technical Pillars / Pilares Técnicos

* **Automation & Scripting:** Uso de Bash Scripting para automação de tarefas repetitivas, coleta de métricas e redução de falhas humanas em rotinas operacionais.
* **Auditability & Logging:** Estruturação de trilhas de auditoria e monitoramento de eventos de sistema para suporte a requisitos de conformidade e segurança.
* **Access Control & System Hardening:** Configuração de permissões rígidas, criptografia de acessos (SSH) e mitigação de vulnerabilidades em servidores.
* **Data Continuity & Backup Governance:** Garantia de integridade e disponibilidade de dados por meio de estratégias automatizadas de backup e retenção.
