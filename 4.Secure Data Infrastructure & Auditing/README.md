# 🔐 Secure Data Infrastructure & Auditing

**PT-BR:**
Este repositório agrupa projetos práticos focados na segurança de infraestruturas de dados, administração de sistemas e automação de auditoria (Bash). Demonstro aqui como construir bases sólidas e resilientes, aplicando políticas de endurecimento de segurança (Hardening - CIS Benchmarks) e gestão segura de arquivos (ECM/GED), garantindo que os servidores, bancos de dados e repositórios de informações críticas estejam protegidos, auditáveis e em total conformidade regulatória.

**EN:**
This repository groups hands-on projects focused on data infrastructure security, system administration, and auditing automation (Bash). Here I demonstrate how to build solid and resilient foundations, applying security hardening policies (CIS Benchmarks) and secure file management (ECM/GED), ensuring that servers, databases, and critical information repositories are protected, auditable, and in full regulatory compliance.

### 🔐 System Integrity & Auditing / Integridade do Sistema & Auditoria:

* **Infrastructure Hardening:** Implementation of security benchmarks (CIS/NIST) to protect data environments and servers.
* **Compliance Automation:** Bash scripting for automated compliance auditing, log collection, and configuration checks.
* **Access Control & Permissions:** Secure SSH configuration, strict Sudoers policies, and File System permissions to protect critical information assets (ECM/GED).
* **Vulnerability Assessment:** Identifying, scanning, and mitigating risks at the infrastructure level to prevent data leakage.
